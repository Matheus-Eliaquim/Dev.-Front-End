# Portfólio Front End
Este repositório concentra meus primeiros projetos em desenvolvimento Front End.
A proposta é registrar minha evolução enquanto aprendo e aplico conceitos fundamentais de HTML, CSS e JavaScript.

🎯 Objetivo
Construir uma base sólida em desenvolvimento web, praticando desde a estruturação de páginas até a criação de interações simples.
Cada projeto aqui reflete um passo no meu aprendizado.

🛠️ Tecnologias em foco
HTML5: marcação semântica e organização de conteúdo

CSS3: estilização, responsividade e layout com Flexbox e Grid

JavaScript: lógica básica, manipulação do DOM e interatividade

📂 Estrutura
Cada projeto possui seu próprio diretório

Incluo comentários e pequenas descrições para contextualizar decisões

O código é mantido de forma simples e legível, priorizando clareza
