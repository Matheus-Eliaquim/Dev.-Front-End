# 📐 Calculadora de Fita LED, Perfis e Fontes

Este projeto tem como objetivo automatizar cálculos relacionados ao uso de fitas LED, perfis de alumínio e fontes de alimentação, otimizando tempo e reduzindo erros em projetos de iluminação.

---

## 🚀 Tecnologias Utilizadas

- **HTML5** → Estrutura base da página  
- **CSS3** → Estilização de componentes e responsividade  
- **JavaScript ES6** → Manipulação da DOM, aplicação de lógica de programação e cálculos  

---

## 📌 Observações

- **Compatibilidade**: O projeto não possui suporte para dispositivos móveis, pois os usuários que o utilizam trabalham exclusivamente em computadores ou notebooks.  

---

## 📝 Sobre o Projeto

Este projeto nasceu de uma necessidade prática na empresa onde trabalho, especializada em projetos de arquitetura voltados para iluminação (residencial, comercial e hotelaria).  

No dia a dia, utilizamos bastante **fitas LED**, que exigem cálculos detalhados para:  
- Determinar a quantidade de rolos de fita necessários.  
- Maximizar o uso de perfis de alumínio.  
- Calcular a potência das fontes de alimentação.  
- Definir quantas fontes serão utilizadas em cada projeto, seguindo os padrões dos fabricantes.  

Atualmente, esses cálculos são feitos manualmente, o que abre margem para erros em projetos grandes e consome muitas horas de trabalho dos vendedores e projetistas.  

---

## 💡 Solução Proposta

A calculadora foi desenvolvida para **automatizar** esse processo.  
Na página, o usuário pode configurar:  
- Potência da fita LED.  
- Tamanho do rolo de fita a ser utilizado.  
- Tamanho do perfil de alumínio.  
- Coeficiente de Segurança.  
- Padrão de distancia para o calculo da quantidade de fontes.  

Assim, o sistema gera os cálculos automaticamente, reduzindo erros e otimizando o tempo de conferência de materiais.
